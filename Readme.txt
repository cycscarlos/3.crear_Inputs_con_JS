
Javascript Learning!

Seriado #3.                   

Contenido:

 "Como obtener el valor escrito en un INPUT y luego manipularlo con Javascript"

Temática:

1. Se creará un INPUT y un Botón con Javascript (sin usar HTML)
2. Se creará una constante de scope Global para validar la entrada del INPUT
3. Se utilizarán las sentencias condicionales IF y ELSE para validar lo obtenido
como "value" del INPUT
4. Se utilizarán las sentencias Substr, toLowerCase y toUpperCase para manipular
la respuesta que se le dará al INPUT
