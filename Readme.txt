Contenido: "Como obtener el valor escrito en un INPUT y luego mainipularlo con Javascript"
------------------

1. Se creará un INPUT y un Botón con Javascript (sin usar HTML)
2. Se creará una constante de scope Global para validar la entrada del INPUT
3. Se utilizará las sentencia IF y ELSE para validar lo obtenido como VALUE del INPUT
4. Se utilizará las sentencias SUBSTR, toLowerCase y toUpperCase para manipular la respuesta que se le dará al INPUT
